# 🏥 Hospital Management System

## 📌 Overview

Hospital Management System is a web-based application designed to manage hospital operations such as patient records, appointments, and data handling.

---

## ✨ Features

* Patient record management
* Appointment scheduling
* Doctor and staff management
* Organized data system
* User-friendly interface

---

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js
* Database / Services: Firebase (if used)

---

## ▶️ How to Run

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd Backend
npm install
npm start
```
